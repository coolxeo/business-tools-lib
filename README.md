# BusinessToolsComponents
Example project that creates a Angular Library and consume it in an app including Storybook

# How to run
```bash
npm i
npm start
```

# Storybook
```bash
npm run storybook
```

# Build Library
Work in progress, for now:
```bash
npm run build-lib
```
